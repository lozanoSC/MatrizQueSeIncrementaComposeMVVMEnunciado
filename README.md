# MatrizQueSeIncrementaComposeMVVMEnunciado

En este repositorio tienes una App simple que visualiza con compose una matriz de forma que la hacer clic en una de sus celdas se incrementa en 1.

En el ejemplo la aplicación está en package org.example.project pero tu tendrás que adaptarlo, si fuera necesario,  a tu proyecto Intellij. También es posible que necesites algun otro cambio.
